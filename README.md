# 🧮 Calculadora em Python com Tkinter

Uma calculadora simples desenvolvida em Python utilizando a biblioteca Tkinter para criar a interface gráfica.

## 📸 Preview

A aplicação possui:

- Interface gráfica moderna
- Operações básicas:
  - Soma
  - Subtração
  - Multiplicação
  - Divisão
- Botão de limpar
- Resultado em tempo real

---

# 🚀 Tecnologias utilizadas

- Python 3
- Tkinter

# ▶️ Como executar

## 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/calculadora-python.git
```

## 2. Entre na pasta do projeto

```bash
cd calculadora-python
```

## 3. Execute o programa

```bash
python main.py
```
